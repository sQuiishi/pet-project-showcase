## 3. Поведение страницы (логика)

| User Action                       | Условие           | Что делает страница                         | Результат / Navigation               |
| --------------------------------- | ----------------- | ------------------------------------------- | ------------------------------------ |
| Открытие страницы                 | —                 | Вызывает функцию получения карточек игроков | FUNC-MARKET-GET-PLAYERS-CARDS-001    |
| Ввод текста в Search Bar          | —                 | Вызывает функцию поиска                     | FUNC-GLOBAL-SEACRH-001               |
| Нажатие "Фильтры"                 | —                 | Открывает / закрывает панель фильтров       | Обновление UI                        |
| Изменение фильтров                | —                 | Обновляет параметры фильтрации              | Ожидание применения                  |
| Нажатие "Применить"               | параметры валидны | Вызывает функцию с параметрами фильтрации   | FUNC-MARKET-GET-PLAYERS-CARDS-001    |
| Нажатие "Сбросить"                | —                 | Сбрасывает параметры фильтрации             | Повторный вызов функции              |
| Изменение параметра сортировки    | —                 | Обновляет параметры сортировки              | FUNC-MARKET-GET-PLAYERS-CARDS-001    |
| Изменение направления сортивровки | —                 | Обновляет направление сортировки            | FUNC-MARKET-GET-PLAYERS-CARDS-001    |
| Переключение страницы             | —                 | Передаёт параметр `page`                    | FUNC-MARKET-GET-PLAYERS-CARDS-001    |
| Нажатие "Профиль"                 | —                 | Переход на страницу профиля игрока          | Навигация на PAGE-PLAYER-PROFILE-001 |
| Нажатие "TG"                      | —                 | Переход по `telegram_link`                  | Открытие внешней системы             |

## 4. Состояния страницы

| State            | Когда возникает        | Что видит пользователь          |
| ---------------- | ---------------------- | ------------------------------- |
| Default          | Есть данные            | Список карточек игроков         |
| Empty            | players_count = 0      | Сообщение об отсутствии игроков |
| Validation Error | Некорректные параметры | Ошибка валидации                |
| Internal Error   | Ошибка выполнения      | Сообщение об ошибке             |

## 5. Правила формы Feedback

| Правило           | Описание                                                  |
| ----------------- | --------------------------------------------------------- |
| Источник данных   | Данные получаются через FUNC-MARKET-GET-PLAYERS-CARDS-001 |
| Видимость игроков | Отображаются только игроки с `is_market_visible = true`   |
| Фильтрация        | Применяется только при передаче параметров                |
| Сортировка        | Применяется после фильтрации                              |
| Пагинация         | Обязательна при большом количестве данных                 |
| Карточка игрока   | Должна соответствовать `market_player_card`               |
| Telegram          | Ссылка используется как внешний переход                   |

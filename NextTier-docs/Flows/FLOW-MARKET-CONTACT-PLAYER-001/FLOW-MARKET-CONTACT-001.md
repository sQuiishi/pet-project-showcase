## 4. Бизнес-правила Flow

- Система MUST вызывать `FUNC-MARKET-GET-PLAYER-CARDS-001` при открытии страницы маркета 
- `FUNC-MARKET-GET-PLAYER-CARDS-001` MUST получать данные игроков из БД
- `FUNC-MARKET-GET-PLAYER-CARDS-001` MUST формировать карточки игроков для отображения на маркете
- Система MUST отображать сформированные карточки игроков на странице маркета
- Пользователь MAY применять фильтры к списку игроков
- Пользователь MAY применять сортировку к списку игроков
- При изменении фильтров система MUST повторно вызывать `FUNC-MARKET-GET-PLAYER-CARDS-001`
- При изменении сортировки система MUST повторно вызывать `FUNC-MARKET-GET-PLAYER-CARDS-001`
- `FUNC-MARKET-GET-PLAYER-CARDS-001` MUST возвращать данные игроков с учётом выбранных фильтров и сортировки
- Пользователь MUST иметь возможность выбрать игрока из списка карточек
- Пользователь MUST иметь возможность перейти к профилю игрока
- Пользователь MUST иметь возможность перейти к связи через Telegram напрямую из карточки игрока
- Пользователь MUST иметь возможность перейти к связи через Telegram из профиля игрока
- Система MUST открыть Telegram-ссылку выбранного игрока при выборе действия `Contact via Telegram`

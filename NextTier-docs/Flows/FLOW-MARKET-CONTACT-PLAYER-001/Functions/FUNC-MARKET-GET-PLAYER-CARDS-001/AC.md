# AC-MARKET-GET-PLAYERS-CARDS-001

## 1. Общая информация о критериях приёмки

| Name     | Description                                                                |
| -------- | -------------------------------------------------------------------------- |
| ID       | AC-MARKET-GET-PLAYERS-CARDS-001                                            |
| Описание | критерии приёмки для функции получения карточек игроков на странице market |

## 2. Таблица сценариев

| AC ID | Сценарий                                                                   | Given (Начальное состояние)                                                                                                                      | When (Действие)                                        | Then (Ожидаемый результат)                                                                                                                                                                                                                          |
| ----- | -------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| AC-1  | успешное получение карточек игроков без параметров фильтрации и сортировки | пользователь находится на странице market; в системе есть хотя бы один игрок с `is_market_visible=true`                                          | система выполняет запрос на получение карточек игроков | система MUST вернуть список `market_player_cards`; система MUST вернуть `players_count`, соответствующий количеству карточек                                                                                                                        |
| AC-2  | успешное получение карточек игроков с параметрами фильтрации               | пользователь находится на странице market; в системе есть игроки с `is_market_visible=true`; переданы валидные параметры фильтрации              | система выполняет запрос на получение карточек игроков | система MUST применить переданные параметры фильтрации; система MUST вернуть только карточки игроков, соответствующие всем условиям фильтрации; система MUST вернуть `players_count`, соответствующий количеству карточек                           |
| AC-3  | успешное получение карточек игроков с параметрами сортировки               | пользователь находится на странице market; в системе есть игроки с `is_market_visible=true`; переданы валидные параметры сортировки              | система выполняет запрос на получение карточек игроков | система MUST применить переданные параметры сортировки; система MUST вернуть отсортированный список `market_player_cards`; система MUST вернуть `players_count`, соответствующий количеству карточек                                                |
| AC-4  | успешное получение карточек игроков с параметрами фильтрации и сортировки  | пользователь находится на странице market; в системе есть игроки с `is_market_visible=true`; переданы валидные параметры фильтрации и сортировки | система выполняет запрос на получение карточек игроков | система MUST сначала применить фильтрацию; система MUST затем применить сортировку к отфильтрованному списку; система MUST вернуть итоговый список `market_player_cards`; система MUST вернуть `players_count`, соответствующий количеству карточек |
| AC-5  | возврат пустого списка карточек игроков                                    | пользователь находится на странице market; подходящие игроки отсутствуют                                                                         | система выполняет запрос на получение карточек игроков | система MUST вернуть `market_player_cards=[]`; система MUST вернуть `players_count=0`; система MAY вернуть `message` для отображения пользователю                                                                                                   |
| AC-6  | исключение непубличных игроков из результата                               | пользователь находится на странице market; в системе есть игроки с `is_market_visible=true` и `is_market_visible=false`                          | система выполняет запрос на получение карточек игроков | система MUST включить в результат только игроков с `is_market_visible=true`; система MUST NOT включать в результат игроков с `is_market_visible=false`                                                                                              |
| AC-7  | возврат карточек игроков по контракту dto                                  | пользователь находится на странице market; в системе есть хотя бы один подходящий игрок                                                          | система формирует `market_player_cards`                | каждая карточка игрока MUST соответствовать контракту `market_player_card` и содержать данные, определённые dto                                                                                                                                     |
| AC-8  | ошибка валидации параметров запроса                                        | пользователь находится на странице market; переданы некорректные параметры фильтрации и/или сортировки                                           | система выполняет запрос на получение карточек игроков | система MUST отклонить запрос и MUST вернуть `validation_error`                                                                                                                                                                                     |
| AC-9  | внутренняя ошибка выполнения функции                                       | пользователь находится на странице market; во время выполнения функции произошла внутренняя ошибка                                               | система выполняет запрос на получение карточек игроков | система MUST завершить выполнение функции с ошибкой `internal_error`                                                                                                                                                                                |

- В критериях приемки оставлены только те сценарии, которые отражают поведение, реализуемое разработчиком.
- Клиентские ошибки, автоматические ошибки сервера и внутренние технические сбои, не требующие отдельной ручной обработки, в критерии не включены.

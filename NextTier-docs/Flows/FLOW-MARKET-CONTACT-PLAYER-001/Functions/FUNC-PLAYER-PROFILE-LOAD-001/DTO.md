# DTO-PLAYER-PROFILE-LOAD-001
## 1. Общая идея документа

| Name                 | Description                                                                    |
| -------------------- | ------------------------------------------------------------------------------ |
| ID                   | DTO-PLAYER-PROFILE-LOAD-001                                                    |
| Назначение документа | описывает входные, системные и выходные данные функции загрузки профиля игрока |

## 2. Input Data — Входные данные

Назначение: Данные, которые передаются системой-клиентом для выполнения функции.

| Data Element | Назначение           | Обязательное | Может быть пустым | Правила / ограничения | Пример |
| ------------ | -------------------- | ------------ | ----------------- | --------------------- | ------ |
| id           | идентификатор игрока | Да           | Нет               | string                | 1002   |




## 3. System Data — Данные, формируемые системой

Назначение: Данные, которые не передаются извне, а определяются системой в процессе выполнения функции.

| Data Element   | Назначение                                     | Когда формируется           | Возможные значения | Пример  |
| -------------- | ---------------------------------------------- | --------------------------- | ------------------ | ------- |
| player_profile | объект профиля игрока                          | при формировании результата | object             | { ... } |
| profile_exists | Признак существования профиля игрока в системе | При проверке игрока по `id` | `true`, `false`    | true    |

## 4. Output Data — Данные, возвращаемые системой

Назначение: Данные, которые возвращаются после успешного выполнения функции.

| Data Element   | Назначение                                                        | Когда возвращается      | Возможные значения | Пример  |
| -------------- | ----------------------------------------------------------------- | ----------------------- | ------------------ | ------- |
| player_profile | Профиль выбранного игрока для отображения страницы Player Profile | При успешном выполнении | object             | { ... } |
### player_profile — Контракт профиля игрока
#### Основная информация

| Data Element       | Назначение                        | Тип    | Пример                                     |
| ------------------ | --------------------------------- | ------ | ------------------------------------------ |
| id                 | Идентификатор игрока в системе    | string | `1002`                                     |
| nickname           | Никнейм игрока                    | string | `s1mple`                                   |
| avatar             | Ссылка на аватар игрока           | string | `https://cdn.example.com/avatar.jpg`       |
| telegram_link      | Ссылка для перехода в Telegram    | string | `https://t.me/player_contact`              |
| faceit_profile_url | Ссылка на профиль игрока в Faceit | string | `https://www.faceit.com/en/players/s1mple` |
#### Персональные данные игрока

| Data Element     | Назначение                           | Тип     | Пример        |
| ---------------- | ------------------------------------ | ------- | ------------- |
| age              | Возраст игрока                       | integer | `18`          |
| primary_role     | Основная игровая роль                | string  | `awper`       |
| secondary_role   | Дополнительная игровая роль          | string  | `rifler`      |
| looking_for_team | Признак того, что игрок ищет команду | boolean | `true`        |
| prime_time       | Предпочтительное игровое время       | string  | `18:30-20:30` |
#### recent_matches — Последние 5 матчей игрока

| Data Element   | Назначение                     | Тип           | Пример             |
| -------------- | ------------------------------ | ------------- | ------------------ |
| recent_matches | Список последних матчей игрока | array<object> | `[ {...}, {...} ]` |

##### recent_match — Контракт одного матча

| Data Element      | Назначение                    | Тип     | Пример                                       |
| ----------------- | ----------------------------- | ------- | -------------------------------------------- |
| match_finished_at | Дата и время завершения матча | integer | `1774627147000`                              |
| score             | Итоговый счёт матча           | string  | `13 / 5`                                     |
| kills             | Количество убийств            | integer | `14`                                         |
| deaths            | Количество смертей            | integer | `12`                                         |
| assists           | Количество ассистов           | integer | `9`                                          |
| adr               | Средний урон за матч          | number  | `84.8`                                       |
| kd_ratio          | Показатель K/D за матч        | number  | `1.17`                                       |
| kr_ratio          | Показатель K/R за матч        | number  | `0.78`                                       |
| headshots_percent | Процент убийств в голову      | integer | `43`                                         |
| mvps              | Количество MVP                | integer | `3`                                          |
| map               | Название карты                | string  | `de_mirage`                                  |
| map_avatar        | Ссылка на изображение карты   | string  | `https://cdn.example.com/maps/de_mirage.png` |


## 5. Result Data — Результаты выполнения

| Outcome type | Result / Error name      | Что возвращается / происходит                                                       | Когда возникает                                              |
| ------------ | ------------------------ | ----------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| Success      | player_profile_loaded    | Система возвращает объект `player_profile`                                          | Профиль игрока по переданному `id` успешно найден и загружен |
| Success      | player_profile_not_found | Система не возвращает объект профиля и возвращает признак отсутствия профиля игрока | Игрок с переданным `id` не найден                            |
| Error        | validation_error         | Система отклоняет запрос                                                            | `id` не передан или передан в некорректном формате           |

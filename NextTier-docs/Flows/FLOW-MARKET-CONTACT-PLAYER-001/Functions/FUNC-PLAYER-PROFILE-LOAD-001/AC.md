# AC-PLAYER-PROFILE-LOAD-001

## 1. Общая информация о критериях приёмки

| Name     | Description                                                     |
| -------- | --------------------------------------------------------------- |
| ID       | AC-PLAYER-PROFILE-LOAD-001                                      |
| Описание | Критерии приёмки для функции загрузки профиля выбранного игрока |

## 2. Таблица сценариев

| AC ID | Сценарий                         | Given (Начальное состояние)                          | When (Действие)                              | Then (Ожидаемый результат)                                                                                                                                                               |
| ----- | -------------------------------- | ---------------------------------------------------- | -------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| AC-1  | Успешная загрузка профиля игрока | Передан валидный `id`, профиль игрока существует     | Система выполняет запрос на загрузку профиля | Система MUST найти профиль игрока; система MUST сформировать объект `player_profile`; система MUST вернуть данные профиля игрока; система MUST вернуть данные только по переданному `id` |
| AC-2  | Профиль игрока не найден         | Передан валидный `id`, профиль игрока отсутствует    | Система выполняет поиск профиля              | Система MUST вернуть признак отсутствия профиля (`player_profile_not_found`); система MUST NOT возвращать объект профиля                                                                 |
| AC-3  | Ошибка валидации входных данных  | `id` не передан или передан в некорректном формате   | Система валидирует входные данные            | Система MUST отклонить запрос; система MUST вернуть `validation_error`                                                                                                                   |
| AC-4  | Внутренняя ошибка выполнения     | Передан валидный `id`, но источник данных недоступен | Система выполняет загрузку профиля           | Система MUST вернуть `internal_error`; система MUST NOT возвращать частично сформированный профиль                                                                                       |



- В критериях приемки оставлены только те сценарии, которые отражают поведение, реализуемое разработчиком.
- Клиентские ошибки, автоматические ошибки сервера и внутренние технические сбои, не требующие отдельной ручной обработки, в критерии не включены.

# 1. Общая информация

Данный документ описывает данные, входящие в данную функцию и исходящие из нее.

# 2. Входные данные

Назначение: Данные, которые передаются системой-клиентом для выполнения функции.

| Элемент          | Тип данных    | Обязательно? | Описание                                       | Правила / ограничения                                       |
| ---------------- | ------------- | -------------| ---------------------------------------------- | ----------------------------------------------------------- |
| telegram         | string        | N            | Ссылка на телеграм                             | "nickname" \| "@nickname" \| "t.me/nickname"                |
| prime_time       | time          | N            | Промежуток времени, когда пользователь активен |                                                             |
| primary_role     | enum          | N            | Основная роль в команде                        | IGL, AWPer, Riffler, Support, Entry fragger, Lurker, Flex   |
| avatar           | file          | N            | Файл аватара                                   | File size <= 10 MB, форматы: jpg, jpeg, png, webp           |

# 3. Данные, формируемые системой

Назначение: Данные, которые не передаются извне, а определяются системой в процессе выполнения функции.

| Элемент            | Тип данных | Когда формируется | Описание                                                  |
| ------------------ | ---------- | ----------------- | --------------------------------------------------------- |
| avatar_url         | string     | В конце функции   | Ссылка на аватар                                          |
| telegram_link      | string     | В конце функции   | Ссылка на профиль телеграм                                |

# 4. Данные, возвращаемые системой

Назначение: Данные, которые возвращаются после выполнения функции.

| Элемент        | Тип данных | Когда возвращается | Описание                               |
| -------------- | ---------- | ------------------ | -------------------------------------- |
| success        | boolean    | Всегда             | Результат выполнения операции          |
| message        | string     | Всегда             | Сообщение о результате выполнения      |
| item           | object     | Код 200            | Измененная запись                      |
| details        | object[]   | Код 400            | Детали о полях, не прошедших валидацию |

Структура item

| Элемент          | Тип данных    | Описание                                            |
| ---------------- | ------------- | --------------------------------------------------- |
| id               | int           | Идентификатор профиля                               |
| avatar_url       | string        | Ссылка на аватар                                    |
| telegram_link    | string        | Ссылка на профиль телеграм                          |
| nickname         | string        | Никнейм игрока на Faceit                            |
| birth_date       | date          | Дата рождения                                       |
| primary_role     | string        | Основная роль в команде                             |
| search_goal      | string        | Цель поиска                                         |
| prime_time       | time          | Промежуток времени, когда пользователь активен      |

# 5. Результаты выполнения

| HTTP Status  | Outcome type | Result / Error name | Что возвращается / происходит                |
| ------------ | ------------ | ------------------- | -------------------------------------------- |
| 200          | Success      | -                   | Возвращается обновленная информация о записи |
| 400          | Error        | validation_error    | Система отклоняет запрос                     |
| 403          | Forbidden    | forbidden           | Система отклоняет запрос                     |

## 1. Общая информация

В этом файле описаны основные типы данных, используемые в рамках процессов с тренировочными матчами (праками).

### 2.5. Класс practice_match_delete_reason

Структура:

| Поле        | Тип данных | Описание                                         |
| ----------- | ---------- | ------------------------------------------------ |
| id          | int        | Идентификатор объекта                            |

Константы:

| id | const_name  | name         | by_user | description                     |

| 3	 | NOT_FOUND   | not found    | true    | Не нашли соперников             |
| 4	 | CANCELLED   | cancelled    | true    | Отмена матча по личным причинам |


## 3. Сущности
## 3.1. practice_match

| Поле             | Тип данных                    | Обязательно? | Пример                   | Описание |
| ---------------- | ----------------------------- | ------------ | ------------------------ | -------- |
| profile_id       | string                        | Y            | 1                        | ID пользователя, который создал прак |
| game_time        | timestamp                     | Y            | 1774872000               | Дата начала прака |
| game_format      | game_format                   | Y            | 3                        | Формат игры |
| maps             | [ game_map ]                  | Y            | [1, 5, 2]                | Карты, на которых будет играться прак |
| avg_elo_team     | int                           | Y            | 2500                     | Среднее ЭЛО команды игрока, создавшего прак. Это значение не высчитывается, юзер указывает сам |
| avg_elo_opponent | int                           | N            | 3000                     | Желаемый ЭЛО противника |
| game_host        | game_host                     | Y            | 2                        | Какая платформа будет хостить прак |
| server_region    | server_region                 | N            | 1                        | Желаемый регион (для пинга) |
| is_active        | boolean                       | Y            | true                     | Активность (актуальность) прака |
| delete_reason    | practice_match_delete_reason  | N            | 1                        | Причина деактивации (удаления) прака |
| created_at       | timestamp                     | Y            | 1774872000               | Дата создания прака |
| updated_at       | timestamp                     | N            | 1774872000               | Дата изменения данных о праке |
| deleted_at       | timestamp                     | N            | 1774872000               | Дата деактивации (удаления) прака |
